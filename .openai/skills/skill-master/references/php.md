# PHP

## Detection signals
- `composer.json`
- `artisan` (Laravel) or `bin/console` (Symfony)

## Source roots
- `app/`, `src/`, `tests/`

## Pattern indicators
- controllers/services/repositories
- DTO/resource classes
- migrations and seeders
- `PHPUnit`

## Key paths
- `app/` or `src/`
- `database/migrations/`
- `tests/`
