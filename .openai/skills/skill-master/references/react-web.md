# React (Web)

## Detection signals
- `package.json` with `react`
- `vite.config.*`, `webpack.config.*`, `next.config.*`

## Source roots
- `src/`, `app/`, `pages/`

## Pattern indicators
- `useState`, `useEffect`, `useMemo`
- `createContext`, `useContext`
- `createSlice`, `configureStore`
- `*.test.*`, `vitest`, `jest`

## Key paths
- `src/components/`
- `src/pages/` or `app/`
- `src/hooks/`
- `src/services/`
