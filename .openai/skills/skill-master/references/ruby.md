# Ruby

## Detection signals
- `Gemfile`, `Gemfile.lock`
- `config/application.rb` (Rails)

## Source roots
- `app/`, `lib/`, `spec/`, `test/`

## Pattern indicators
- MVC controllers/models/services
- ActiveRecord models and migrations
- Sidekiq workers
- RSpec or Minitest

## Key paths
- `app/`
- `db/migrate/`
- `spec/` or `test/`
