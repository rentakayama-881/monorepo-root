# Node.js

## Detection signals
- `package.json`
- `tsconfig.json` or `jsconfig.json`

## Source roots
- `src/`, `lib/`

## Pattern indicators
- `express()` / `fastify()` / `nestjs`
- `*.service.*`, `*.controller.*`
- `zod` or `joi` validation
- `jest`, `vitest`, `mocha`

## Key paths
- `src/routes/`
- `src/controllers/`
- `src/services/`
- `src/repositories/`
